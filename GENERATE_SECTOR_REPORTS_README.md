# 行业板块指数报告批量生成说明

## 概述

已创建10个脚本文件，用于分批生成86个行业板块指数的分析报告。

## 文件说明

- `generate_sector_reports.py` - 主程序，用于生成行业报告
- `generate_sector_group_1.sh` ~ `generate_sector_group_10.sh` - 10个分组脚本

## 使用方法

### 方法1：逐个运行脚本（推荐）

```bash
# 运行第1组（9个行业）
./generate_sector_group_1.sh

# 运行第2组（9个行业）
./generate_sector_group_2.sh

# ... 依此类推，直到第10组
./generate_sector_group_10.sh
```

### 方法2：直接使用Python脚本

```bash
# 生成单个行业报告
python3 generate_sector_reports.py --sectors "BK1031"

# 生成多个行业报告
python3 generate_sector_reports.py --sectors "BK1031 BK1015 BK1033"

# 指定输出文件夹
python3 generate_sector_reports.py --sectors "BK1031" --output "reports/my_reports"
```

## 输出位置

报告将保存在 `reports/sector_reports_YYYYMMDD_HHMMSS/` 目录下。

每个行业会生成一个PDF文件，文件名格式：`{行业名称}_{行业代码}_行业指数分析报告.pdf`

## 注意事项

1. 确保已安装所有依赖：`pip install -r requirements.txt`
2. 确保网络连接正常（需要访问AKShare API）
3. 每个脚本运行时间取决于网络速度和数据量，预计每组需要5-15分钟
4. 如果某个行业数据获取失败，脚本会继续处理下一个行业
5. 报告生成完成后，会在控制台显示成功和失败的统计信息
