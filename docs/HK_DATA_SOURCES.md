# 港股数据源说明

## 📊 当前数据源（免费）

系统使用免费数据源获取港股数据，自动降级，无需任何API密钥。

### 1. 新浪财经（主要）
- 支持港股日线
- 依赖少，仅需 `requests`

### 2. 东方财富（备选）
- 支持港股日线
- 数据覆盖面好

### 3. AKShare（备选）
- 支持港股日线
- 支持港股分钟线（1m/5m/15m/30m/60m）

## 🔄 自动降级顺序

```
1. 新浪财经
   ↓ (失败)
2. 东方财富
   ↓ (失败)
3. AKShare（若已安装）
   ↓ (失败)
4. 返回错误
```

## 📝 周期支持

- ✅ 日线：新浪/东方财富/AKShare
- ✅ 分钟线：AKShare（1m/5m/15m/30m/60m）

## ✅ 使用示例

```bash
# 港股分析
python3 github_stock_bot.py --mode manual --stocks "02899 00700"
```
