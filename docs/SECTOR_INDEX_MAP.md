# 行业板块指数代码对照表

本文件列出了所有可查询的行业板块指数代码和名称对照。

## 使用方法

在运行 `github_stock_bot.py` 时，可以通过 `--sector` 参数指定行业：

```bash
# 通过代码查询
python3 github_stock_bot.py --mode manual --stocks "688630" --sector "BK1031"

# 通过名称查询
python3 github_stock_bot.py --mode manual --stocks "688630" --sector "光伏设备"
```

## 数据来源

行业板块指数数据来自 **AKShare** 的 `stock_board_industry_hist_em` 接口（同花顺行业板块）。

## 注意事项

- 行业代码以 `BK` 开头
- 支持通过代码（如 `BK1031`）或名称（如 `光伏设备`）查询
- 如果名称不完全匹配，系统会尝试模糊匹配
- 行业指数数据仅支持日线，不支持分钟线
